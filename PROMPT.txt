# DESCRIPTION DU PROJET
Une plateforme web mettant en relation automatique des étudiants et des entreprises via un moteur de recommandation basé sur le NLP (Traitement du Langage Naturel). Le système analyse le CV PDF de l'étudiant, extrait les compétences, et utilise TF-IDF et la Similarité Cosinus pour calculer un score de pertinence entre le profil et chaque offre de stage existante. C'est 85% plus puissant qu'un annuaire classique.

# STACK TECHNIQUE
- Backend : Python, Django, Django REST Framework (DRF)
- Base de données : PostgreSQL
- Moteur IA (NLP) : scikit-learn (TF-IDF/Cosine), pdfplumber/PyMuPDF (extraction PDF)
- Frontend : React.js (Vite), TailwindCSS, Axios (Architecture Headless)

# PLAN D'IMPLÉMENTATION (Les 5 Phases)
- [Phase 1] Fondations Backend : Architecture Django (Pattern MVT vers API), app `users`, configuration PostgreSQL.
- [Phase 2] Modélisation (Models) : Surcharge User (Student/Company), Modèles de Profils et Offres (InternshipOffer).
- [Phase 3] API REST : ViewSets, Serializers, Authentification JWT.
- [Phase 4] NLP & IA : Parseur de CV PDF (`utils/resume_parser.py`) et moteur de correspondance TF-IDF (`utils/recommender.py`). Endpoint d'API de scoring.
- [Phase 5] Frontend React : Authentification, UI Drag-and-Drop pour les CV, Dashboard avec visualisation des scores de recommandation.
