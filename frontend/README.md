# Plateforme Stages - Frontend

Frontend React pour la plateforme de recommandation de stages.

## Fonctionnalités

- Authentification JWT
- Dashboard étudiants et entreprises
- Recherche et filtrage d'offres
- Gestion des candidatures
- Upload de CV
- Recommandations IA

## Technologies

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Axios

## Installation

```bash
cd frontend
npm install
```

## Développement

```bash
npm run dev
```

Ouvre [http://localhost:5173](http://localhost:5173) dans le navigateur.

## Build

```bash
npm run build
```

## Backend
